<h1>⚡ PDF-BOT — RAG-Based PDF Question Answering System</h1>
<hr></hr>
<br></br>
<h2>Overview</h2>
<p1>PDF-BOT is a fast and efficient Retrieval-Augmented Generation (RAG) application built using Streamlit, FAISS, HuggingFace embeddings, and Groq LLM (Llama 3.1).

It allows users to:

Upload PDF documents 📄 ,
Ask questions 🔍,
Get accurate, context-based answers.</p1>

<hr></hr>
<h2>🧠How it works?</h2>
<p2><ul>1.PDF → Text Extraction</ul>
<ul>2.Text Chunking</ul>
<ul>3.Embeddings Generation (MiniLM)</ul>
<ul>4.FAISS Vector Storage</ul>
<ul>5.Similarity Search</ul>
<ul>6.LLM (Groq) Answer Generation</ul></p2>

<hr></hr>
<h2>✨ Features</h2>
<p1><ul>⚡ Ultra-fast responses using Groq API</ul>
<ul>🧠 Context-aware answers (No hallucination)</ul>

<ul>📋 Auto Key Points extraction</ul>
<ul>📑 Document summarization</ul>
<ul>🎨 Clean and responsive UI</ul>
<ul>💾 Efficient caching for performance</ul></p1>
